OceanSonar.MetricFunction — Type
OceanSonar.MetricFunction <: Function

Called instances dispatch its first argument as Symbol or AbstractString to Model.

E.g.

using OceanSonar
sound_speed isa OceanSonar.MetricFunction
import OceanSonar: sound_speed
sound_speed(::Model{:MyModel}, z::Real) = 1500.0 + exp(-z^2)
sound_speed(Model(:MyModel), 1e3)
sound_speed(:MyModel, 1e3)
sound_speed("My Model", 1e3)

See also: Model.

OceanSonar.@implement_metric_function — Macro
@implement_metric_function function_name

Implements a new MetricFunction <: Function instance ready for Model dispatching.

Specifically:

  1. Defines a concrete struct subtype of MetricFunction.
  2. Defines a const instance of the new subtype with the name function_name.
  3. Adds the new function to the list of metric_functions.

See also: Model, OceanSonar.MetricFunction.

Example

@implement_metric_function sound_speed_profile
sound_speed_profile(::Model{:MyModel}, z::Real) = 1500.0 + exp(-z^2)
sound_speed_profile("My Model", 1e3)
